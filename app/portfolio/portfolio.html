Portfolio

Financial Heatmap

Requirements

A UI for visualizing a financial portfolio. Rectangles are sized to represent a stock's proportion of the portfolio. The color of the rectangles represented the stock's positive or negative contribution to the portfolio.

Portfolios could be grouped into larger sections, which should be represented similarly to stocks. When a section is clicked, the interface should zoom into the clicked section to give greater detail.

Hovering over a particular stock should give a pop-up with more detailed information about the stock.

With this interface, a user should easily be able to navigate and discover the significant contributors and detractors from their portfolio.

Simulation Equalizer

Requirements

An interface for setting various input factors for a simulation. There are 10 variables that can be adjusted on a sliding scale.

The variables have historical values associated with them. Various statistics about the historical values are indicated along with the current setting of the variable and whether or not it is above or below the historical mean.

Any variable can be set to human or simulation control, which should be clearly indicated. After the variables are set and submitted, a bar chart updates to show the results of the simulation.
