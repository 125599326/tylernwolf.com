Description

A Sketch plugin I developed that converts CSV files to data tables and charts in Sketch. The plugin also generates columns of dummy data for data tables and dummy line and bar charts. The plugin is optimal for any data-heavy design applications and can be used as a quick way to prototype and design data visualizations.

Implementing this presented a few UX challenges with consideration for how to work with the Sketch plugin implementation and OS X dialogues to make the plugin as easy to understand and use as possible.

The plugin is steadily growing a user base and was featured in an Invision blog post. It has been downloaded over 1,000 times and is visited hundreds of times a month.

I made a premium version of the plugin with added functionality based the needs of designers on data-heavy projects and data scientists. Many people have found the functionality in the premium version useful. It can be found here.

The source code for the free version of the plugin is available on Github. You can find it here.
