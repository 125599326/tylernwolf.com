Sketch Data Studio (code/tool)

Description

A Sketch plugin I developed that converts CSV files to data tables or charts in Sketch. The plugin also generates columns of dummy data for data tables and dummy line and bar charts. The plugin is optimal for any data-heavy design applications and can be used as a quick way to prototype and design data visualizations.

The plugin is steadily growing a user base and was featured in an Invision blog post (see it here).
