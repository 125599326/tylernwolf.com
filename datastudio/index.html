Sketch Data Studio


Thank you for your interest in Sketch Data Studio!


Sketch Data Studio is a Sketch plugin that enables data importing from CSV files into Sketch compositions. Imported CSV data can be used to generate charts and data tables - a huge time saver that pays for itself.

Chart types currently supported are Bar charts and Line charts in addition to Data Tables. By purchasing a license to Sketch Data Studio, you receive the Premium version of the plugin and will receive future updates to the software free of charge. In addition, you will receive direct customer support should you encounter any issues with your purchase.
